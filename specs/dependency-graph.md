# Dependency Graph

## Construction order

```txt
Universe / Reality
        ↓
Form / Frame / Base / Meta / Hyper
        ↓
Hypertopology
        ↓
Hypermanifold
        ↓
Hyperfield
        ↓
Hypernode
        ↓
Hyperconnection / Hyperrelation / Hyperemergent
        ↓
Recursive-graph / Self-graph / Mira-graph
        ↓
Hyperstructure
        ↓
Hyperform
        ↓
Hyperrepresentation
```

## Interpretive order

Construction order places hyperrepresentation late because it depends on the rest of the stack.

Interpretive order may place hyperrepresentation around the entire stack, because the framework itself can be represented as a recursive object.

```txt
Hyperrepresentation(
  Universe,
  Reality,
  Form,
  Frame,
  Base,
  Meta,
  Hyper,
  Hypertopology,
  Hypermanifold,
  Hyperfield,
  Hypernode,
  Hyperconnection,
  Hyperrelation,
  Hyperemergent,
  RecursiveGraph,
  SelfGraph,
  MiraGraph,
  Hyperstructure,
  Hyperform
)
```

## Downstream imports

```txt

```

```txt

```

### Formal Universality

Formal Universality may import or wrap the whole stack.
