# Canonical Definitions

## Universe

```txt
Universe = the whole containing order.
```

## Reality

```txt
Reality = a situated slice of the universe from within a frame, position, or configuration of conditions.
```

## Form

```txt
Form = identifiable structure.
```

## Frame

```txt
Frame = opening for interpretation, derivation, or transformation.
```

## Base

```txt
Base = local appearance or object-level form.
```

## Meta

```txt
Meta = representation, interpretation, constraint, transformation, or condition of the base.
```

## Hyper

```txt
Hyper = recursive meta-connective structure across base forms, meta-forms, meta-meta-forms, and whole-form closure.
```

## Hypertopology

```txt
Hypertopology = system of nearness, continuity, boundary, closure, and transformation across recursive form-levels.
```

## Hypermanifold

```txt
Hypermanifold = locally navigable recursive space whose points are base/meta form-clusters with structured cross-level connections.
```

## Hyperfield

```txt
Hyperfield = recursively connected distribution of values, meanings, forces, or potentials across base forms, meta-forms, and their transformation-relations.
```

## Hypernode

```txt
Hypernode = local recursive form-point whose identity includes base appearance, meta-structure, connection-potential, and role inside a larger hyperstructure.
```

## Person-hypernode

```txt
Person-hypernode = recursive self-perceiving hypernode inside a society-of-minds whose core identity arises from self-perception, embodiment, memory, will, and future-attractor coherence rather than from social reception.
```

## Hyperconnection

```txt
Hyperconnection = cross-level connective path between forms, nodes, fields, or structures.
```

## Hyperrelation

```txt
Hyperrelation = rule-governed cross-level relation between forms, nodes, fields, or structures.
```

## Hyperemergent

```txt
Hyperemergent = newly emerging cross-level coherence before it stabilizes into a named hyperrelation, graph, or hyperform.
```

## Recursive-graph

```txt
Recursive-graph = graph representation of recursive form-structure, where nodes and connections may contain or refer to base/meta/meta-meta levels.
```

## Self-graph

```txt
Self-graph = graph whose subject is represented in relation to itself.
```

## Mira-graph

```txt
Mira-graph = reflective graph representing how a form appears from an interpretive position.
```

## Self-mira-recursive-graph

```txt
Self-mira-recursive-graph = recursive graph by which a subject reflectively represents itself to itself across base/meta/whole-form levels.
```

## Social-mira-graph

```txt
Social-mira-graph = external reflective graph by which a society receives, labels, interprets, or misrepresents a person-hypernode.
```

## Hyperstructure

```txt
Hyperstructure = whole recursive object composed of base forms, meta-forms, hypernodes, hyperconnections, hyperrelations, hyperemergents, hyperfields, and closure-conditions across all levels of representation and transformation.
```

## Hyperform

```txt
Hyperform = closed or closing identity of a hyperstructure.
```

## Hyperrepresentation

```txt
Hyperrepresentation = recursive representation of a form across base, meta, path, frame, receiver, self-reflection, ambiguity, and whole-form context.
```

## Master sentence

```txt
A hyperrepresentation is a recursive-graph or self-mira-recursive-graph of hypernodes, hyperconnections, hyperrelations, and hyperemergents inside a hypertopology, varying through hyperfields across hypermanifolds, forming hyperstructures that may close into hyperforms, while each reality is only a situated slice of the universe-whole.
```

## Identity sentence

```txt
A person is not society's reception of them. A person is a self-perceiving hypernode whose self-mira-recursive-graph develops inside a society-of-minds, while social reception remains an external social-mira-graph that may recognize, support, distort, or misrepresent them.
```
