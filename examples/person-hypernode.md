# Person-Hypernode Example

## Claim

```txt
A person is not society's reception of them.
```

A person is a recursive self-perceiving hypernode inside a society-of-minds.

Their core identity arises from:

```txt
self-perception
embodiment
memory
will
future-attractor coherence
```

Social reception is not core identity.

Social reception is:

```txt
external social-mira-graph
exo-identity pressure
attempted representation by an outside system
```

It may:

```txt
recognize
support
distort
constrain
misrepresent
```

## Graph distinction

```txt
self-mira-recursive-graph != social-mira-graph
```

### Self-mira-recursive-graph

A self-mira-recursive-graph is a recursive graph by which the person-hypernode reflectively represents itself to itself across base/meta/whole-form levels.

It includes:

```txt
self-perception
memory
embodiment
will
future-attractor
identity-continuity
self-interpretation
repair
```

### Social-mira-graph

A social-mira-graph is an external reflective graph by which society receives, labels, interprets, or misrepresents the person-hypernode.

It includes:

```txt
labels
reputation
stereotypes
roles
institutional categories
audience interpretation
algorithmic compression
projection
misrepresentation
```

## Principle

```txt
Reception is not identity.
Reception is an attempted representation of identity by an outside system.
```
